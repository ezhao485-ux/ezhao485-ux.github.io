/filldrive (virus command)

Fills the player's drive with dummy files. Presumably to stress test the game from crashes due to a full disk.

The danger

If the target doesn't crash this command can be run over and over again. You can make your target's computer create and delete and create and delete. This will harm your target's computer and if the game is open for too long then their computer will get bricked.

History

Bedrock edition

1.2.13 release Accidentally added developer commands to iOS, including /filldrive.

1.2.14 Removed all developer commands.
